# 蔬菜识别与管理系统技术报告

## 1. 项目背景

随着人们健康意识的提升，对蔬菜营养价值的关注度日益增加。然而，许多消费者在购买蔬菜时难以准确识别品种，也不了解其营养价值和烹饪方法。本项目旨在开发一款基于深度学习的蔬菜识别应用，帮助用户快速识别蔬菜品种并获取相关信息。

## 2. 系统架构设计

### 2.1 整体架构

系统采用前后端分离的三层架构：

- **表示层**：Android客户端，负责用户界面展示和交互
- **业务层**：Spring Boot后端，负责业务逻辑处理
- **数据层**：MySQL数据库，负责数据持久化存储
- **AI模块**：PyTorch深度学习框架，负责图像识别

## 5. 深度学习模块设计

### 5.1 数据集准备

- **数据来源**：网络爬取 + 手动收集
- **类别数量**：10类（西红柿、黄瓜、胡萝卜、茄子、菠菜、西兰花、土豆、白菜、青椒、豆角）
- **数据划分**：训练集80%，验证集20%
- **数据增强**：
  - 随机裁剪 (RandomResizedCrop)
  - 随机水平翻转 (RandomHorizontalFlip)
  - 随机旋转 (±15°)
  - 颜色抖动 (亮度、对比度、饱和度)

### 5.2 模型架构

采用迁移学习策略，基于ImageNet预训练的ResNet18模型：

```
输入层: 3×224×224 RGB图像
    ↓
卷积层: Conv2d(3, 64, kernel_size=7, stride=2, padding=3)
    ↓
批归一化: BatchNorm2d(64)
    ↓
激活函数: ReLU
    ↓
池化层: MaxPool2d(kernel_size=3, stride=2, padding=1)
    ↓
残差块 Stage 1: 2个 BasicBlock (64通道)
    ↓
残差块 Stage 2: 2个 BasicBlock (128通道)
    ↓
残差块 Stage 3: 2个 BasicBlock (256通道)
    ↓
残差块 Stage 4: 2个 BasicBlock (512通道)
    ↓
全局平均池化: AdaptiveAvgPool2d(1, 1)
    ↓
展平层: Flatten
    ↓
全连接层: Linear(512, 10)  ← 修改为10类输出
    ↓
输出层: 10类蔬菜概率分布
```

### 5.3 训练策略

| 参数 | 设置 |
|------|------|
| 优化器 | Adam |
| 学习率 | 0.001 |
| 学习率衰减 | 每7个epoch衰减为0.1倍 |
| 损失函数 | CrossEntropyLoss |
| 批大小 | 32 |
| 训练轮数 | 20 |

### 5.4 训练流程

```python
# 1. 加载数据集
dataset = datasets.ImageFolder('vegetable_dataset/train', transform)

    # 训练阶段

    # 验证阶段
    model.eval()
    with torch.no_grad():
        for inputs, labels in val_loader:
            outputs = model(inputs)
            # 计算准确率
    
    # 保存检查点
```

### 5.5 模型转换

将PyTorch模型转换为TorchScript格式，便于Android端部署：


```java
// 2. 图像预处理
private Tensor preprocessImage(Bitmap bitmap) {
    // 调整大小
    Bitmap resized = Bitmap.createScaledBitmap(bitmap, 224, 224, true);
    
    // 归一化
    FloatBuffer buffer = FloatBuffer.allocate(1 * 3 * 224 * 224);
    for (int y = 0; y < 224; y++) {
        for (int x = 0; x < 224; x++) {
            int pixel = resized.getPixel(x, y);
            buffer.put(((pixel >> 16) & 0xFF) / 255.0f);  // R
            buffer.put(((pixel >> 8) & 0xFF) / 255.0f);   // G
            buffer.put((pixel & 0xFF) / 255.0f);          // B
        }
    }
    
    return Tensor.fromBlob(buffer.array(), new long[]{1, 3, 224, 224});
}

// 3. 模型推理
private String recognize(Bitmap bitmap) {
    Tensor inputTensor = preprocessImage(bitmap);
    IValue output = module.forward(IValue.from(inputTensor));
    float[] scores = output.toTensor().getDataAsFloatArray();
    
    // 获取预测结果
    int predictedClass = argmax(scores);
    return classNames[predictedClass];
}
```

### 6.3 Android网络请求封装

```java
public class ApiClient {
    private static final String BASE_URL = "http://192.168.1.100:8080";
    private static OkHttpClient client;
    
    public static void get(String path, ApiCallback callback) {
        Request request = new Request.Builder()
            .url(BASE_URL + path)
            .get()
            .build();
        client.newCall(request).enqueue(new Callback() {
            @Override
            public void onResponse(Call call, Response response) {
                callback.onSuccess(response.body().string());
            }
            @Override
            public void onFailure(Call call, IOException e) {
                callback.onError(e.getMessage());
            }
        });
    }
}
```
